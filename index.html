Token do GitHub

Pra adicionar ou editar produtos por aqui, o painel precisa de um token do GitHub (fica salvo só neste navegador). Crie em github.com/settings/personal-access-tokens/new: Repository access → Only select repositories → claude-price-monitor; Permissions → Contents: Read and write e Actions: Read and write (essa segunda é pro botão "Coletar agora"). Expiração: pode escolher "No expiration".
